Blueprint Skalatra
=========================

---

`Inspired from things I have worked on or seen.`

---

Basic scalatra application blueprint.

Other features:
 - status endpoint
 - cucumber
 - Apache 2.4
 - Async logging and log rotation
 - Swagger


To run application:

    sbt run



To run unit tests

    sbt test



To run cucumber tests

    sbt cucumber
